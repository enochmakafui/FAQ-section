FAQs

What is Frontend Mentor, and how will it help me ?

Frontend Mentor Offers realistic coding challenges to help developers improve their frontend coding skills with projects in HTML, CSS and Javascript.It's suitable for all levels and ideal for portfolio building.

Is Frontend mentor Free?

Yes, Frontend Mentor offers both free and premium coding challenges, with the free option providing access to range of projects suitable for all skills levels.

Can I use Frontend mentor projects for my portfolio?

Yes, you can use projects completed on Frontend Mentor in your portfolio.It's an excellent way to showcase your skills to potential employers!

How can I get help if my stuck on a challenge?

The best place to get help is inside Frontend Mentor's Discord community.There's a help channel there you can ask questions and seek support from other community members.
